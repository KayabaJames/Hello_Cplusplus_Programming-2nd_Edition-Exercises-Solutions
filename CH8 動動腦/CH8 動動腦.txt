(P.8-17)
1. 下列函數的功能是什麼？
void copyArray(int a1[], int s, int a2[], int t, int n)
{
    for(int i = s; i < s + n; i++)
    {
        a2[i] = a1[t];
        t++;
    }
}

Ans: 這個函數的功能是將一個陣列(a1)中的元素內容複製到另一個陣列(a2)中。
具體來說，它從a1的t位置開始，複製n個元素到a2的s位置開始的地方。
這個函數假設a1和a2的大小足夠大，能夠容納複製的元素。
例如：
a1[5] = {1, 2, 3, 4, 5}
a2[6] = {6, 7, 8, 9, 10, 11} --> a2[6] = {1, 2, 3, 4, 5, 11}
