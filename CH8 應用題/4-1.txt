Question: 
請說明下列函式為m, n兩數之最大公因數gcd的遞迴式。
gcd(m, n) = m，若n = 0
            gcd(n, m % n)，若n! = 1

Ans: 
說明：
這是一個使用歐幾里得算法(Euclidean Algorithm)來計算兩個數字的最大公因數(GCD)的遞迴函數。歐幾里得算法是一種非常古老的算法，用於找出兩個整數的最大公因數。

這個算法的基本思想是：兩個整數的最大公因數等於其中較小的數和兩數相除餘數的最大公因數。具體來說，如果我們有兩個數字 m 和 n(假設 m > n)，那麼 m 和 n 的最大公因數就等於 n 和 m % n 的最大公因數。

遞迴函數的定義如下：
1. 如果 n 等於 0，那麼 m 和 n 的最大公因數就是 m。這是因為任何數字和 0 的最大公因數都是該數字本身。
2. 如果 n 不等於 0，那麼我們就遞迴地計算 n 和 m % n 的最大公因數。這是因為 m 和 n 的最大公因數等於 n 和 m % n 的最大公因數。

這個遞迴函數會一直遞迴下去，直到 n 等於 0 為止。在這種情況下，函數就會返回 m，這就是 m 和 n 的最大公因數。
